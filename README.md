# Animated Bento Card

A beautifully animated bento card with smooth entrance and exit animations, built with Next.js and Motion.

![Demo](https://github.com/nabedkhan/animated-bento-card/blob/main/public/demo.png)

## Features

- Smooth entrance and exit animations
- Built with Next.js 15
- Styled with Tailwind CSS
- Animated with Motion (motion.dev)
- TypeScript support
- Responsive design

## Notice

This card is inspired by and recreates a component from [Aceternity UI Pro](https://pro.aceternity.com/products/bento-grids)
